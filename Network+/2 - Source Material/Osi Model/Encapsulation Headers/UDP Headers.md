# UDP Headers

### UDP Header Fields

- **Source Port (16 bits)**  
  Identifies the sending port.

- **Destination Port (16 bits)**  
  Identifies the receiving port.

- **Length (16 bits)**  
  Specifies the total length of the UDP header and data.

- **Checksum (16 bits)**  
  Used for error-checking (optional in IPv4, required in IPv6).
